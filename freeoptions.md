You don’t actually need Skilljar to do what you want. You can build a **very similar “course platform” for free** using YouTube + a few tools layered together.

# 🧠 The simplest “Skilljar-like” setup (FREE)

## 🎥 1. Host your course videos

Use:

* YouTube

👉 Key trick:

* Upload videos as **Unlisted** (so only your students can access)
* Organize them into **Playlists = your course modules**

✔ YouTube is ideal because it gives **unlimited hosting + global delivery for free** ([TechRadar][1])

---

## 🧱 2. Build your “course portal” (the Skilljar feel)

You need a simple frontend where users:

* See modules
* Click lessons
* Track progress (basic)

### Option A (easiest): Notion site

* Create pages → Module 1, Module 2
* Embed YouTube videos
* Share as a website

👉 Looks clean + no coding

---

### Option B (better): WordPress + LMS plugin

Use:

* WordPress
* Free LMS plugin like:

  * Tutor LMS
  * LearnPress

👉 These let you:

* Structure lessons
* Lock content (progression)
* Add quizzes
* Create user accounts

✔ These plugins are **free and fully functional for basic courses** ([fatlabwebsupport.com][2])

---

## 🧪 3. Add quizzes (Skilljar-like engagement)

Use:

* Google Forms

👉 Embed inside your course:

* Quiz after each module
* Collect answers
* Gate progress manually (or via LMS plugin)

---

## 🔐 4. Control access (important)

### Free ways:

* Private YouTube + share only inside your site
* Password protect your site
* Use email list (send links manually)

👉 If you want payments later:

* Add:

  * Stripe
  * PayPal
